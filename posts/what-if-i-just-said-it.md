graf-after--p">I imagine them reading it and cringing.<br>I imagine them thinking, “What is this?”<br>I imagine myself shrinking after I press send.</p><p name="b0d8" id="b0d8" class="graf graf--p graf-after--p">And yet…</p><p name="5658" id="5658" class="graf graf--p graf-after--p">It feels like something I have to say.</p><p name="2ec3" id="2ec3" class="graf graf--p graf-after--p">Not to get something from them.<br>Not to be praised or validated.<br>But because it’s true.<br>Because I’ve carried these thoughts inside me for too long,<br>and maybe they deserve to live outside me for once.</p><p name="8275" id="8275" class="graf graf--p graf-after--p">You know that feeling when a kid writes something for class — <br>a poem, a letter, a little paragraph — <br>and they’re nervous about showing it to their parents?<br>Not because it’s wrong,<br>but because it’s real.</p><p name="ebaa" id="ebaa" class="graf graf--p graf-after--p">That’s how I feel.</p><p name="0bca" id="0bca" class="graf graf--p graf-after--p">Like I’ve scribbled something honest in a notebook,<br>and now I’m walking around with it in my hand,<br>unsure if I should pass it to the people it’s meant for.</p><p name="7476" id="7476" class="graf graf--p graf-after--p">And I wonder…<br>don’t others feel this way, too?</p><p name="c91f" id="c91f" class="graf graf--p graf-after--p">Don’t others hesitate before saying something kind?<br>Something vulnerable?<br>Something that cracks the surface we all wear?</p><p name="be23" id="be23" class="graf graf--p graf-after--p">It’s so easy to keep these things hidden.<br>To lock them away in the name of pride or fear or timing.</p><p name="5bf7" id="5bf7" class="graf graf--p graf-after--p">But I don’t want to hide anymore.<br>Even if my voice trembles.<br>Even if my timing’s off.<br>Even if they don’t say anything back.</p><p name="e4c7" id="e4c7" class="graf graf--p graf-after--p">Because saying it — <br>speaking from that raw, awkward, trembling place — <br>matters to me.</p><p name="2ead" id="2ead" class="graf graf--p graf-after--p">Even if no one else sees it.<br>Even if it lands weird.<br>Even if I blush with embarrassment after.</p><p name="d8e6" id="d8e6" class="graf graf--p graf-after--p">Maybe that’s okay.</p><p name="0779" id="0779" class="graf graf--p graf-after--p">Maybe that’s human.</p><p name="3131" id="3131" class="graf graf--p graf-after--p graf--trailing">And maybe…<br>this time, I’ll say it anyway.</p>
